Proyecto de asistente de conductor desarrollado para el Guadalahacks 2026 del Tec de Monterrey Campus Guadalajara.

**Requisitos**
- Node.js 16+.


**Instalar dependencias**

```
pnpm install
```

**Ejecutar en modo desarrollo**

```
pnpm dev
```

En http://localhost:5173.
